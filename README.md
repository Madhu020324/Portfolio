# Madhu Yamini Nallamothu - Portfolio

A professional, responsive portfolio website built with modern web technologies. This project showcases my skills, experience, and projects as a Senior Software Engineer.

## Tech Stack
-   **React**: UI Library
-   **TypeScript**: Type Safety
-   **Vite**: Fast Build Tool

## Features
-   **Responsive Design**: Optimized for all devices.
-   **Interactive Sections**:
    -   **Hero**: Introduction with social links.
    -   **Experience**: Timeline of work history.
    -   **Skills**: Categorized technical skills.
    -   **Projects**: Showcase of recent work with links and tech stacks.
    -   **Education**: Academic background.
    -   **Contact**: Functional contact form and details.

## Running Locally

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
